# Blink-it-sales-analysis-project
📊 Blinkit Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project presents an interactive sales analysis dashboard for Blinkit (India’s last-minute delivery app). The dashboard is built using Power BI to visualize key business insights such as sales performance, outlet distribution, product categories, and customer ratings.

🎯 Objectives
Analyze total sales performance
Understand outlet distribution by size, type, and location
Evaluate product category performance
Track customer satisfaction using ratings
Provide data-driven insights for business decisions
📈 Key Features
🔹 KPI Cards
Total Sales: $1.20M
Average Sales: $141
Number of Items: 8523
Average Rating: 3.9
🔹 Visualizations
Outlet Establishment Trend – Sales trend over years
Fat Content Analysis – Comparison of Low Fat vs Regular products
Sales by Item Type – Performance of different product categories
Outlet Size Distribution – Small, Medium, High outlets
Outlet Location Analysis – Tier 1, Tier 2, Tier 3 cities
Outlet Type Table – Grocery store vs supermarkets comparison
🧩 Filters / Slicers
Outlet Location
Outlet Size
Outlet Type

These filters allow dynamic exploration of the data.

🛠️ Tools & Technologies
Power BI Desktop – Data visualization
DAX (Data Analysis Expressions) – Calculations & measures
Excel / CSV Dataset – Data source
📂 Dataset Description

The dataset includes:

Sales data
Product categories
Outlet details (size, type, location)
Customer ratings

Explore insights through interactive visuals
